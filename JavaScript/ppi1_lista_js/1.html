O que é JavaScript?

A motivação para criar o JavaScript foi de que antes só tinha um HTML estático exibindo elementos na tela, mas não era possível validar ou atualizar informações,
ou seja, não conseguia-se interagir com as ações do usuário como os eventos de clique. Assim, o JavaScript surge para solucionar estas questões e possibilitar
diversas interações que antes não era possível. Além disso, o a diferença de rodar o JavaScript no navegador e no servidor é de que no navegador estará
rodando-o no sandbox, e no servidor não ocorre isso. E o sandbox no navegador isola os sites criados de acessar diretamente os sistemas operacionais e
arquivos do computador para uma maior segurança. E ainda o document é o que representa o HTML, então ao utilizar o JavaScript e quiser por exemplo
chamar algum elemento do HTML precisa primeiro chamar document na frente; já o console é para depurar/testar códigos e o alert exibe uma mensagem de
alerta ao usuário.
