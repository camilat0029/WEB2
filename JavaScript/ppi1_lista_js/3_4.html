Como declarar variável

Usamos let ou var para declarar variável

var nome; //cria uma variável global com identificador;
let idade; //cria uma variável local com um identificador idade

Tratamento de evento em JavaScript

Tratamos evento em JS usando addEventListener que vincula uma função a um evento específico do DOM.

Exemplo:

document.addEventListener('click', function(){ console.log("Clique detectado") })

Eventos Comuns disponíveis nos Navegadores

  • click - evento de click
  • mouseover - evento de mouse sobre o elemento
  • mouseout - evento de mouse fora o elemento
  • mouseenter - evento de mouse entrando no elemento
  • keypress - evento da tecla pressionada caracteres visiveis ASCII
  • keydown - evento da tecla pressionada
  • ketup - evento de tecla solta
  • submit - evento de encio de formulário da tag form
  • DOMContentLoaded - evento disparado quando o documento HTML é completamente carregado e parseado
  • change - evento disparado quando o valor de um elemento de formulário é alterado
