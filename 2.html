Regra CSS

A regra CSS é criada para estilizar as páginas Web, sendo que ela é aplicada em elementos HTML(tags, identificadores, classes). Um exemplo seria criar uma regra CSS para aplicar na tag do parágrafo uma fonte de maior ou mudar a cor do seu texto.


Neste caso, a tag de paragrafo selecionada no CSS seria o seletor. Então, os seletores identificam qual componente do HTML receberá a regra, eles podem ser: o * (aplicará a regra em tudo), um # (identificador único), .class (uma classe, que pode se referir a varios componentes ao mesmo tempo, desde que tenham a classe igual), entre outros seletores. Porém, também pode-se combinar esses seletores para criar uma regra, assim a que tiver mais prioridade será a regra aplicada, por exemplo se criar uma regra com para a tag de um elemento e também criar uma regra para a classe deste mesmo elemento, a regra da classe irá prevalecer.


Na regra CSS também tem-se as propriedades e seus valores. Em que a propriedade será oque será aplicado no seletor e o valor que dirá como ou quanto será essa propriedade. Exemplo: a propridade de width (para determinar a largura do componente) com valor de 120px, a propridade de background-color (cor do fundo) com valor de Blue e assim por diante.


Exemplos:


    #exemplo1 {

        color: rgb(120, 1, 231);

    }

    Exemplo de regra para mudar cor do texto.

    Exemplo de pseudo classe:

    .exemplo2:hover {
        background-color: rgb(255, 126, 148);
        font-size: 1.3em;
    }
