Protocolo HTTP

O HTTP(Protocolo de Transferência de Hipertexto) é usuado para carragar páginas web usando os links de hipertexto.
Este protocolo funciona em um modelo de comunicação que utiliza uma arquitetura Cliente-Servidor. Assim,
quando o cliente faz uma solicitação ao servidor, este protocolo é utilizado nesta comunicação

Requisição

O cliente ao fazer uma requisição ao servidor, envia-se uma mensagem HTTP, que podem ter ações como
por exemplo GET ou POST. E o retorno do servidor após processá-la é uma response HTTP.

No caso dos servidores WEB, o retorno é um código HTML como uma página.

Request e response na Web
