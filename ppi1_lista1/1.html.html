Web e Internet

Diferença entre Internet e Web

A internet é uma rede de computadores e funciona utilizando a arquitetura (Cliente - Servidor), já a web funciona como uma forma de gerenciar e compartilhar informações. Assim para a web funcionar ela precisa de internet, mas a internet não precisa necessariamente da web para funcionar, pois pode oferecer outros serviços além da web, como e-mail, ssh, entre outros

História da Web e Internet


Contexto

(1947-1991): Período em que ocorreu a Guerra Fria, ela não teve um confronto direto, porém as grandes
potências estavam competindo entre si, o que ocasionou grandes inovações tecnológicas


Sistemas de Comunicação Centralizados

  • Sistemas de comunicação militares evoluíram bastante;
  • Alta vulnerabilidade;
  • Dependiam de redes de telefonia ou eram baseados em radares e rádios;
  • Todos eram sistemas centralizados;
  • Se tivessem poucas estações, o serviço poderia ser facilmente interrompido.

Internet

A internet surge em 1969 com a criação da ARPANET, nos Estados Unidos,
que inicialmente era um projeto militar e acadêmico

ARPANET

Advanced Research Projects Agency Network
  1. Financiada pela ARPA;
  2. Criada em 1969;
  3. Foi uma iniciativa para interctar laboratórios de pesquisa;
  4. Era uma rede de troca de informações;
  5. No início ligou 4 universidades e possibilitou o compartilhamento de informações;
  6. Nos anos 80, mudou o protocolo NCP para TCP/IP, permitindo desta forma que computadores de diferentes redes pudessem se comunicar;

Serviços na Internet

1988:
Rede de internet chega ao Brasil.
1990:
Serviços de rede encontrados eram o E-MAIL, MIRC, gospher e ftp.
1994:
Rede liberada para uso privado e comercial.
1995:
Internet já estava consolidada como principal fonte de comunicação.

WEB

Em 1989, Tim Berners-Lee inventou a World Wide Web. Propondo assim um sistema de gerenciamento de informações utilizando hipertextos, isso possibilitava uma maior facilidade no compartilhamento das informações. Já no ano seguinte (1990) ele desenvolveu o protocolo HTTP que é utilizado pela Web, assim ela funciona no modelo Cliente-Servidor, tendo as requisições HTTP e suas respostas.
Para saber mais sobre este protocolo vá para HTTP
